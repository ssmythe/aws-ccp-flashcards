# AWS CCP Flashcards

A simple, single-page HTML flashcard app for studying **AWS Certified Cloud Practitioner (CLF-C02)** exam services.

## Features
- Flip cards to reveal summaries and service categories  
- Search and filter by AWS category  
- Shuffle and reset for randomized study sessions  
- 100% client-side (no external dependencies beyond Bootstrap)

## Usage
1. Open [`aws_ccp_flashcards.html`](./aws_ccp_flashcards.html) in any modern web browser.  
2. Click **Flip** to reveal service details.  
3. Use the **Search**, **Category**, **Shuffle**, and **Reset** controls to study interactively.

## Source
Services and summaries derived from the official  
**[AWS Certified Cloud Practitioner Exam Guide (CLF-C02)](https://aws.amazon.com/certification/certified-cloud-practitioner/)**.
